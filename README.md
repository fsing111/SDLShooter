# SDL Shooter

一个使用 C++ 和 SDL2 开发的 2D 射击游戏。

这个项目主要用于学习：

- SDL2 游戏开发
- C++ 面向对象设计
- 场景管理（Scene System）
- UTF-8 中文输入处理
- 游戏循环与碰撞检测
- CMake 工程结构
